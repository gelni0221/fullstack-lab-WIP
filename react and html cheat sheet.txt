2ï¸âƒ£ useState

What it is:

A React hook that lets your component remember values (state).

Purpose:

Store dynamic data that can change and cause the component to re-render.

Key points:

Returns a value and a function to update it.

Updating the state automatically re-renders the component with the new value.

Example concept:

Track if the user is logged in (true/false).

Track a counter that increases when a button is clicked.
